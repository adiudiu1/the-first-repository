# Research Checklist

Classify every item from user statements or exact file sources. Do not supply missing values.

## Problem

- Scheduling setting and operational constraints
- Dynamic events
- Optimization objectives
- Feasibility conditions

## Stochastic arrivals

- Arrival process, parameters, and units
- Horizon or stopping rule
- Scenario generation and reproducibility

## Disjunctive graph

- Node meaning
- Conjunctive-edge meaning and direction
- Disjunctive-edge meaning and direction
- Node, edge, graph, and global features
- Graph update timing

## MDP or environment

- State and observation
- Action and feasibility mask
- Transition timing
- Reward and accumulation
- Discounting
- Terminal and truncation conditions

## Learning method

- Algorithm
- Function approximator
- Optimization configuration
- Exploration mechanism
- Checkpoint and evaluation policy

## Baselines

- Exact implementation source
- Parameterization
- Information available to each method
- Runtime or decision-budget matching

## Experiments

- Instance source or generator
- Training, validation, and evaluation separation
- Scale and arrival regimes
- Seeds and matched-instance identifiers
- Measures and direction of improvement
- Compute budget and stopping conditions
- Ablations, sensitivity, and generalization

## Reproducibility

- Dependency declaration
- Configuration source
- Randomness controls
- Structured result schema
- Logs, checkpoints, failure records, and repository revision
